Geração antecipada de PIX

Como preencher?

  • Valor: É o valor dos pagamentos em PIX que devem ser gerados.
  • Qtd Min: É o limíte mínimo para a geração dos pagamentos antecipados PIX. Se o total de registros for menor do que este valor a rotina irá gerar novos registros de pagamento até o limite máximo.
  • Qtd Max: É o máximo de registros de pagamento que a chave e valor podem ter armazenados. Ao se atingir este limite a rotina para de gerar novos registros, e iniciará novamente quando a quantidade atingir o mínimo.

Os PIX Antecipados serão usados para deixar o processo de venda mais rápido, pois PIX de paramento já estará gerada no momento em que o cliente criar uma compra!

Crie valores iguais aos que serão utilizados nas campanhas que utilizam a chave PIX {{$ctrl.data.key}}.

Esta rotina também é útil para os casos em que a Cartos sair do ar por qualquer motivo!
